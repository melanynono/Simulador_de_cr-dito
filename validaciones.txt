3. PROMPT PARA ESTRUCTURA DE FUNCIONES DE VALIDACIÓN
================================================================================

"Crea una arquitectura modular de funciones de validación en JavaScript:

FUNCIONES GENÉRICAS (reutilizables):
1. limpiarErrores() - Elimina todos los mensajes de error y clases CSS
2. mostrarError(idInput, idError, mensaje) - Muestra un error específico
3. validarCampoVacio(valor, nombreCampo) - Retorna error si está vacío
4. validarNumero(valor, nombreCampo) - Retorna error si no es número
5. validarPositivo(valor, nombreCampo) - Retorna error si no es positivo
6. validarNoNegativo(valor, nombreCampo) - Retorna error si es negativo
7. validarRango(valor, nombreCampo, min, max) - Retorna error si fuera de rango
8. validarMaxDigitos(valor, nombreCampo, maxDigitos) - Retorna error si excede dígitos

FUNCIONES ESPECÍFICAS (por campo):
1. validarIngresos(valor) - Aplica todas las reglas para ingresos
2. validarEgresos(valor) - Aplica todas las reglas para egresos
3. validarEgresosVsIngresos(ingresos, egresos) - Validación relacional
4. validarMonto(valor) - Aplica todas las reglas para monto
5. validarPlazo(valor) - Aplica todas las reglas para plazo
6. validarTasaInteres(valor) - Aplica todas las reglas para tasa

FUNCIÓN PRINCIPAL:
validarFormulario() - Coordina todas las validaciones y retorna true/false

ARQUITECTURA:
- Cada función específica llama a las funciones genéricas en secuencia
- Si una validación falla, retorna el error inmediatamente (early return)
- Si todas pasan, retorna null (sin error)
- La función principal recorre todos los campos y acumula errores
- Retorna true solo si TODOS los campos son válidos"

6. PROMPT PARA EXPRESIONES REGULARES DE VALIDACIÓN
================================================================================

"Define las expresiones regulares necesarias para validar tipos de datos:

REGEX PARA NÚMEROS CON DECIMALES:
Patrón: /^-?\d+(\.\d+)?$/
Explicación:
- ^ = inicio de cadena
- -? = signo negativo opcional
- \d+ = uno o más dígitos
- (\.\d+)? = punto decimal seguido de dígitos (opcional)
- $ = fin de cadena
Uso: Validar ingresos, egresos, monto, tasa de interés

REGEX PARA NÚMEROS ENTEROS:
Patrón: /^\d+$/
Explicación:
- ^ = inicio de cadena
- \d+ = uno o más dígitos
- $ = fin de cadena
Uso: Validar plazo en años

VALIDACIÓN DE MÁXIMO DÍGITOS:
Código:
const valorEntero = valor.split('.')[0].replace('-', '');
if (valorEntero.length > maxDigitos) {
    // Error
}
Explicación:
- split('.')[0] = obtener parte entera
- replace('-', '') = remover signo negativo
- length = contar caracteres
- Comparar con máximo permitido"
